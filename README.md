# 🍔 Cardápio Fullstack

Aplicação fullstack para gerenciamento de itens de um cardápio, permitindo cadastrar e visualizar produtos

---

## 🚀 Tecnologias

### 🔹 Backend

* Java
* Spring Boot

### 🔹 Frontend

* React
* Vite

### 🔹 Banco de Dados

* PostgreSQL

---

## ✨ Funcionalidades

* Cadastro de itens do cardápio
* Listagem de produtos
* Integração completa entre front-end e back-end

---

## 🎯 Objetivo
Desenvolver uma aplicação fullstack para consolidar conhecimentos em desenvolvimento web, utilizando Spring Boot no backend e React no frontend, além de praticar integração com banco de dados e criação de APIs REST.

---

## 🧠 Conceitos aplicados

* API REST
* CRUD
* Integração com banco de dados
* Consumo de API no front-end

---

## ▶️ Como rodar o projeto

### 🔹 Backend

```bash
cd backend
./mvnw spring-boot:run
```

### 🔹 Frontend

```bash
cd frontend
npm install
npm run dev
```

## 💡 Aprendizados

* Desenvolvimento de aplicações fullstack
* Criação de APIs com Spring Boot
* Integração entre front-end e back-end
* Manipulação de dados com PostgreSQL
